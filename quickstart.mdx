## Setup your development
Clone the SDK:
```
git clone https://github.com/robot-learning-co/trlc-sdk.git && cd trlc-sdk
```
Create a Conda environment and install the SDK:
```
conda create -n trlc-sdk python=3.10 && conda activate trlc-sdk
pip install -e .
```
You are ready to go!
